# 🚗 ESP32-CAM Surveillance Car

> An IoT-based Wi-Fi surveillance rover using the ESP32-CAM for real-time video streaming, remote navigation, and gas detection.

## 📖 Overview

The ESP32-CAM Surveillance Car is an IoT-enabled robotic rover designed for real-time monitoring and surveillance applications. The system allows users to remotely control the rover over Wi-Fi while receiving live video from the ESP32-CAM module. An MQ-2 gas sensor continuously monitors the surrounding environment, and a buzzer provides an alert whenever hazardous gas levels are detected.

This project demonstrates the integration of embedded systems, IoT, robotics, and wireless communication into a single surveillance platform.

## ✨ Features

- 📹 Real-time video streaming using ESP32-CAM
- 🌐 Wi-Fi based remote control
- 🎮 Wireless movement control
- 🎥 Pan-Tilt camera mechanism
- 🧪 MQ-2 Gas detection
- 🔔 Automatic buzzer alert
- ⚡ Battery-powered mobile platform
- 🤖 Compact surveillance rover
